# Logs Folder

Place execution logs here:

- `android-unit.log` from `./gradlew :app:testDebugUnitTest :shared:testDebugUnitTest`
- `android-assemble.log` from `./gradlew :app:assembleDebug`
- `ios-xcodebuild.log` from iOS simulator/device build commands
- Optional: any backend or integration logs used in triage
