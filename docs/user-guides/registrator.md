# Памятка регистратора: пользователи и фильтры

## Цель
Быстро находить пользователей, создавать новые учетные записи и корректно работать с доступами.

## Порядок работы
1. Откройте `Пользователи`.
2. Примените фильтры: поиск, роль, группа, сортировка.
3. Для нового пользователя нажмите `+`.
4. Выберите режим создания и заполните ФИО, роли и группу.
5. В ручном режиме задайте логин и пароль вручную.
6. После создания скопируйте логин и пароль из карточки результата.
7. Для существующего пользователя используйте `Сменить пароль` (автогенерация).

## Частые ошибки
- Пустые обязательные поля при создании.
- Неверно выбрана группа или роль.
- Ожидание ручного reset-пароля для существующего пользователя (он авто).

## Что делать при отказе
1. Уточните текст ошибки и повторите действие.
2. Проверьте, что логин не занят.
3. Если ошибка постоянная, зафиксируйте шаги и передайте администратору.
