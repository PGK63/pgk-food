# Памятка куратора: работа с группами и дубликатами

## Цель
Понимать порядок перевода/переименования группы и безопасно работать при дубликатах названий.

## Порядок работы
1. Проверьте, есть ли у аккаунта права `REGISTRATOR` или `ADMIN` для групповых операций.
2. При переносе группы сначала определите исходную группу по `ID`.
3. Если есть одинаковые названия (например, две `ИСП-31`), выбирайте цель только по `ID`.
4. Дополнительно сверяйте куратора и число студентов в группе.
5. Переносите студентов в целевую группу.
6. После переноса назначайте куратора целевой группе.
7. Удаляйте исходную группу только после проверки, что в ней нет студентов.

## Частые ошибки
- Выбор группы только по названию при наличии дубликатов.
- Попытка выполнить перенос без нужной роли.
- Удаление группы до завершения переноса студентов.

## Что делать при отказе
1. Проверьте права текущего пользователя.
2. Повторите операцию, сверяя `ID` групп.
3. Если ошибка не уходит, сохраните `ID` и время операции для разбора.
